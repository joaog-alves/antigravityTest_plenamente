📖 Passo a passo para acessar e testar o site
  1. O servidor backend (Spring Boot) está ativo localmente em http://localhost:8080.
  2. O banco de dados H2 (em memória) foi semeado com usuários padrão para testes.
  3. Clique em qualquer perfil na seção de Acesso Rápido no rodapé para preencher os dados de login instantaneamente.
  4. Se deseja inspecionar as tabelas no banco de dados, acesse http://localhost:8080/h2-console (use jdbc:h2:mem:plenamentedb e deixe a senha em branco).
